# Credible Follow-Up Package

Use this folder to action Task #11 (Follow up with Credible).

## Step 1 — Locate the original thread (do this first)

1. Open Zoho Mail → search inbox for: `credible.com`, `Credible`, `partner`, `publisher`
2. **Also check the Junk / Spam folder** — partner program replies frequently land there
3. If you applied through their partner portal (publishers.credible.com or similar), log in and check application status
4. Note the date of your last outbound email — this determines which template below to send

## Step 2 — Pick the right template

| Situation | Template to use |
|---|---|
| They replied with follow-up questions | `02-answers-to-questions.md` |
| Silent for 2+ weeks, no questions | `01-gentle-checkin.md` |
| They declined | (no template — see "If declined" below) |
| They went silent after asking questions | `03-second-followup.md` |

## Step 3 — Send from the right address

Send from **chris@apruvee.com** (matches the contact info on the live site). Keep the original thread — reply in-thread rather than starting a new one whenever possible, so they have full context.

## Step 4 — Log the outcome

After sending, update `.local/tasks/followup-credible.md` (or the project task) with:
- Date sent
- Which template used
- Any reply received

---

## If declined

Ask for the specific reason (traffic source, content depth, compliance, volume, vertical fit). Common Credible objections and how to address them:

- **"Not enough traffic yet"** — Ask if they offer a deferred/conditional approval for sites in soft launch, with a re-review at 90 days. Mention you're investing in SEO content and paid acquisition (Google/Meta) starting now.
- **"Need to see compliance in action"** — Offer a screen-share walkthrough of the lead form, TCPA consent step, and disclosures.
- **"Vertical mismatch"** — Apruvee is squarely personal loans / debt consolidation, which is Credible's core. Re-confirm vertical and ask which specific concern they had.
